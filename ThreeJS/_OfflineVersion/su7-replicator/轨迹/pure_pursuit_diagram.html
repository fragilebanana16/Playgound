Pure Pursuit — 车走圆弧到目标点

推导过程

① 弦长公式（圆几何） distance = 2R · sin(α) → R = distance / (2·sin(α))
② 自行车模型（直角三角形 OPQ） tan(θ) = L / R → R = L / tan(θ)
③ 联立①② L / tan(θ) = distance / (2·sin(α))
④ 解出转向角 tan(θ) = 2·L·sin(α) / distance
θ = atan( 2·L·sin(α) / distance )
⑤ 代码实现 Math.atan2(2·wheelbase·sin(α), distance)
